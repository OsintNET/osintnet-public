# OsintNET

OsintNET is a web-based OSINT platform that combines domain intelligence, search-result discovery, identity investigation, image intelligence, image authenticity analysis, breach exposure checks and passive website risk scanning.

Website: https://www.osintnet.com  
Tools overview: https://www.osintnet.com/osint-tools

## What OsintNET Provides

- Domain Analyzer: DNS, WHOIS/RDAP, ASN/BGP, SSL, reverse IP, SPF, DMARC, DNSSEC and HTTP evidence.
- SERP Scanner: search-result collection, domain discovery and investigation leads.
- Deep OSINT: public-signal investigation for emails, usernames, phone numbers, domains and companies.
- Image Intel: EXIF metadata, OCR, visual clues, reverse-search evidence and geolocation hypotheses.
- Image Authenticity Scanner: AI image detection, metadata inspection, generator traces and forensic signals.
- Email Breach Check: public exposure, paste clues, GitHub metadata and mail-security signals.
- Website Risk Scanner: passive website security audit for authorized targets.

## Access Model

OsintNET is designed with a free core workflow. Some advanced features may become paid or require higher limits in the future, but the public tools should remain useful for basic OSINT workflows.

## Safety And Ethics

OsintNET is intended for lawful OSINT, security research, journalism, brand protection, fraud investigation and defensive analysis. Users should only scan websites, accounts, images or identifiers they are authorized to investigate.

The Website Risk Scanner is passive by design. It does not brute-force credentials, exploit vulnerabilities, attack login forms or run aggressive port scans.
