# Roadmap

This roadmap is directional and may change as OsintNET evolves.

## Current Focus

- Improve image intelligence and geolocation evidence quality.
- Expand Image Authenticity Scanner with stronger AI-generated image detection and clearer forensic explanations.
- Improve Deep OSINT username and email workflows with better confidence scoring.
- Add cleaner exports for analysts, legal teams and AI-assisted review.
- Strengthen SEO and public documentation for each OSINT workflow.

## Planned Improvements

- More source-linked evidence cards.
- Better result deduplication and confidence scoring.
- Team accounts and saved investigations.
- API access for selected tools.
- Monitoring and alerting for domains, breach exposure and website risk.
- More transparent data-handling controls.

## Possible Paid Features

The free workflow should remain useful. Future paid features may include:

- Higher query limits.
- Bulk scans.
- Professional PDF reports.
- Saved history.
- API access.
- Team workspaces.
- Monitoring and alerts.
- Premium enrichment providers.
