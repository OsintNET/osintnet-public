# Contributing

Thank you for your interest in improving OsintNET.

This repository is currently used for public feedback, documentation and roadmap tracking. The full production source code is not published here.

## Good Contributions

- Bug reports with clear reproduction steps.
- Broken source or link reports.
- Suggestions for new OSINT sources.
- Ideas for safer workflows.
- Documentation improvements.
- False-positive or false-negative examples for image, breach or risk tools.

## Before Opening An Issue

Please include:

- Which OsintNET tool you used.
- The target type, without exposing private data.
- Expected result.
- Actual result.
- Screenshots when useful.
- Browser and device information if UI-related.

## Responsible Use

Do not submit private personal data, credentials, illegal material or targets you are not authorized to investigate.
